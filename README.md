# ⏰ Digital Clock

A modern and responsive Digital Clock built using HTML, CSS, and JavaScript.

## 📌 Features

- Real-time clock updates every second
- 12-hour and 24-hour format support
- Responsive design
- Clean and modern UI
- Beginner-friendly JavaScript project

## 📂 Project Structure

```text
Digital-Clock/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 Learning Objectives

This project helps developers learn:

- DOM Manipulation
- JavaScript Date Object
- setInterval() Function
- CSS Styling
- Responsive Design
